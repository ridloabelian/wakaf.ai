Everything You Need to Manage Waqf

Arsitektur hybrid: Cloudflare Workers di edge + VPS di Jakarta untuk core services

AI Gateway

Multi-provider LLM dengan KIMI sebagai primary, fallback ke OpenRouter, Anthropic, dan OpenAI. Dengan caching dan rate limiting.

10 Skema LSP BWI

Semua skema sertifikasi nazhir tersedia: SS.001 sampai SS.010. Terintegrasi dengan sistem ERP dan workflow otomatis.

Edge + Core Hybrid

Cloudflare Workers untuk routing dan AI di edge. VPS Tencent Cloud Jakarta untuk database, ERP, dan monitoring.

Real-time Monitoring

Grafana dashboard untuk monitoring semua services. Health check otomatis setiap 15 menit via GitHub Actions.

Enterprise Security

Auth via Cloudflare Access, rate limiting, DDoS protection, dan enkripsi end-to-end untuk semua data wakaf.

Multi-Agent System

23 AI agents bekerja bersama untuk mengelola berbagai aspek wakaf: ERP, penerimaan, penjagaan, penyaluran, dan keuangan.
